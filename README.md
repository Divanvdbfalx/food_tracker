# Weight Gain Tracker (Supabase)

A Streamlit app for tracking bodyweight and calorie intake across a 16-week weight-gain plan, backed by Supabase.

## Features

- Separate bodyweight logging and calorie logging
- Multiple calorie entries per day with meal tags (`breakfast`, `lunch`, `snack`, `dinner`, `other`)
- Dashboard metrics for weight trend and average daily calories vs target
- Charts for daily bodyweight, 7-day rolling weight average, and summed daily calories vs target
- Mobile-friendly responsive layout for phone screens
- CSV downloads generated from Supabase data

## Setup

```bash
pip install -r requirements.txt
```

## Supabase Setup

1. Create a Supabase project.
2. In SQL Editor, run:

```sql
create table if not exists public.bodyweight_log (
  id bigint generated by default as identity primary key,
  date date not null,
  weight_kg numeric not null,
  notes text default ''
);

create table if not exists public.calorie_log (
  id bigint generated by default as identity primary key,
  date date not null,
  time text not null,
  meal_tag text not null,
  calories integer not null,
  notes text default ''
);
```

3. In Supabase Project Settings -> API, copy:
- Project URL
- `anon` public key

4. Add Streamlit secrets.

Local (`.streamlit/secrets.toml`):

```toml
SUPABASE_URL = "https://YOUR_PROJECT_REF.supabase.co"
SUPABASE_KEY = "YOUR_SUPABASE_ANON_KEY"
```

For Streamlit Community Cloud, add the same keys in App Settings -> Secrets.

## Run

```bash
streamlit run app.py
```

## Notes

- The app requires `SUPABASE_URL` and `SUPABASE_KEY` to start.
- For production, add Row Level Security and policies once auth is introduced.
